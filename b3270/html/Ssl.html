ssl Indication

Change in SSL state.

AttributeAlways present?Purpose
secureyestrue or false, connection is secure
verifiednotrue or false, host certificate is verified
sessionnosession details
host-certnohost certificate details

The verified, session and host-cert attributes are present only if secure is true.

Example:
