charset Indication

Indicates support for a particular host code page. One or more are sent, only at initialization. Each indication may include one more more aliases with the attributes alias1, alias2, etc.

AttributeAlways present?Purpose
nameyesCanonical code page name
alias1noFirst alias for the code page
alias2noSecond alias for the code page

Example: 

<charset name="cp1140" alias1="us-euro"/>
