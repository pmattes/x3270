screen Indication

A screen update. Screen elements contain one or more nested row elements.

Example: 

 <screen>
  <row row="1">
   <char col="1" text="abcde" fg="red">
   <char col="12" text="zxcvb" fg="blue">
   <attr col="46" bg="green" count="4">
  </row>
 </screen>
